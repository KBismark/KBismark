### 📦 NoscDB — Hybrid Node.js Database Engine

<table>
<tr>
<td width="55%">

**The first-ever hybrid database system written in pure JavaScript** running in a Node.js environment. NoscDB uniquely blends relational and non-relational database practices — with tables, columns, and document-style rows — into a fast, asynchronous key-value storage system.

**Key Highlights:**
- 🔀 Hybrid: relational + document-based in one
- ⚡ Non-blocking async operations
- 📈 Horizontally scalable across multiple servers
- 🟢 Beginner-friendly API (`getRow`, `updateRow`…)
- 🗃️ Runs in-process — no separate server needed

**Tech Stack:**
